О документе: итоговое тестовое ТЗ на MVP CRM_14.

## 1. Назначение системы

CRM_14 предназначена для учебного моделирования CRM-системы, которая позволяет работать с лидами, контролировать стадии воронки и получать базовую аналитику.

## 2. Пользователи

- менеджер;
- аналитик;
- руководитель отдела продаж.

## 3. Функциональные требования

### FR-001. Выбор роли

Система должна позволять выбрать роль пользователя без полноценной авторизации.

### FR-002. Создание лида

Система должна позволять менеджеру и РОП создать лида.

### FR-003. Просмотр списка лидов

Система должна показывать список лидов с учётом роли.

### FR-004. Канбан

Система должна показывать лиды по стадиям в канбан-доске.

### FR-005. Смена стадии

Система должна позволять менять стадию лида по бизнес-правилам.

### FR-006. Возврат стадии

Система должна позволять менеджеру запросить возврат, а РОП — одобрить или отклонить его.

### FR-007. История лида

Система должна показывать историю стадий лида.

### FR-008. Импорт

Система должна поддерживать импорт CSV/XLSX.

### FR-009. Экспорт

Система должна поддерживать экспорт Excel/CSV.

### FR-010. Фильтры

Система должна поддерживать фильтры по владельцу, источнику и периоду.

### FR-011. KPI

Система должна считать конверсию и среднюю длительность стадий.

## 4. Нефункциональные требования

### NFR-001. Производительность

Отчёт должен формироваться на тестовом наборе из 100 лидов за приемлемое время.

Принятое допущение: в финальной версии можно указать точный порог, например ≤ 2 сек.

### NFR-002. Учебная безопасность

Система не должна хранить реальные персональные данные.

### NFR-003. Ролевая доступность

Интерфейс должен ограничивать действия пользователя по выбранной роли.

### NFR-004. Идемпотентность импорта

Повторный импорт не должен создавать дубли по `lead_uid`.

### NFR-005. Трассируемость

Изменения стадий должны отображаться в истории лида.

## 5. Ограничения MVP

- нет регистрации;
- нет логина/пароля;
- нет телефонии;
- нет email;
- нет персональных данных;
- нет суммы сделки;
- нет промышленного деплоя.
