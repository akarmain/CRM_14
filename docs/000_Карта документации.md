# Карта документации CRM_14

|**Версия**|**Статус**|**Дата создания**|**Дата обновления**|
|---|---|---|---|
|v0.2-test|Draft|2026-04-03|2026-04-27|

О документе: карта документации объясняет, какие артефакты входят в проект CRM_14 и как читать единый комплект.

Для кого: для команды, преподавателя и проверяющего.

Основано на: структуре демонстрационного документа и фактических артефактах проекта CRM_14.

## Назначение карты

Карта нужна, чтобы быстро понять, где в документации находятся:

- описание проекта;
- требования;
- роли;
- бизнес-процессы;
- модель данных;
- API;
- тестовые сценарии;
- демо-сценарий;
- план сдачи.

Хотя текущая версия собрана в один Markdown-файл, внутри используются имена документов, как если бы они лежали в отдельных папках репозитория.
