# Contributing

## Getting Started
- Use `docker compose up --build` to run the full stack.
- Frontend runs at `http://localhost:3000`, backend at `http://localhost:8000`.

## Development Workflow
- Keep changes focused and follow the layered backend structure.
- Add or update tests when changing backend behavior.
- Run formatters before opening a PR (`ruff format .` and `npm run format`).

## Pull Requests
- Provide a short summary and testing notes.
- Include screenshots for UI changes.
